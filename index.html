About Virtual Learning Journey Builder

A powerful, intuitive tool designed to create immersive, multi-scene interactive learning experiences without coding.

Key Features:
  • Upload and link 360° panoramas
  • Interactive hotspots with text, video, and portals
  • Linear quiz checkpoints (Transit Logic)
  • ✨ Gemini AI integration for auto-filling content and generating challenges (WIP)
  • 1-Click Export to a standalone HTML file
